# flac-picture-encoder

encode an image file (jpg, png) to embed it as a cover art of an audio file (vorbis, opus, flac).

```sh
# development
pnpm dev
pnpm test
pnpm build
pnpm -s cli < misc/test.jpg
pnpm e2e

# release
pnpm release
```
